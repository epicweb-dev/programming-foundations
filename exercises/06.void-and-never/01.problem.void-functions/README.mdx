# Void Functions

<EpicVideo url="https://www.epicweb.dev/workshops/programming-foundations/6-void-and-never/void-functions" />

👨‍💼 Many functions in real applications don't return values—they perform
actions. These are called **side effects**, and they return `void`.

Examples of side effects:

- Logging information
- Updating the UI
- Sending data to a server
- Playing a sound

🐨 Open <InlineFile file="index.ts" /> and:

1. Create a `logInfo` function that logs a message with "[INFO]" prefix
2. Create a `logError` function that logs with "[ERROR]" prefix
3. Create a `logWithTimestamp` function that includes the current time

💰 Use `Date` to get the current time and include it in the log.

💰 Void functions don’t return a value.
