# Booleans and Logical Operators

<EpicVideo url="https://www.epicweb.dev/tutorials/programming-foundations/3-primitive-types/booleans-and-comparisons" />

👨‍💼 Our product catalog needs to track whether items are available and combine
multiple conditions. Booleans and logical operators are essential for making
decisions in code.

You've already learned that comparisons return booleans. Now let's work with
boolean values directly and combine them with **logical operators**.

## Logical Operators

```ts
const both = a && b // true if BOTH are true
const either = a || b // true if EITHER is true
const opposite = !a // inverts the boolean
```

Use parentheses to group expressions and control order of operations:

```ts
const result1 = (a && b) || c // && happens first, then ||
const result2 = a && (b || c) // || happens first, then &&
const result3 = (a || b) && c // || happens first, then &&
```
