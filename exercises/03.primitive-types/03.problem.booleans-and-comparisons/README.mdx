## Logical Operators

```ts
const both = a && b // true if BOTH are true
const either = a || b // true if EITHER is true
const opposite = !a // inverts the boolean
```

Use parentheses to group expressions and control order of operations:

```ts
const result1 = (a && b) || c // && happens first, then ||
const result2 = a && (b || c) // || happens first, then &&
const result3 = (a || b) && c // || happens first, then &&
```
