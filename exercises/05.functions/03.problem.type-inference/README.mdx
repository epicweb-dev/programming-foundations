# Type Inference

<EpicVideo url="https://www.epicweb.dev/tutorials/programming-foundations/5-functions/type-inference" />

👨‍💼 TypeScript is smart about inferring types. You don't always need to write
them explicitly—especially for return types.

```ts
function add(a: number, b: number) {
	return a + b // TypeScript knows this returns number
}
```

But TypeScript **cannot** infer parameter types (in most cases). You must
specify them:

```ts
// ❌ Error: Parameter 'a' implicitly has 'any' type
function add(a, b) {
	return a + b
}

// ✅ Correct
function add(a: number, b: number) {
	return a + b
}
```
