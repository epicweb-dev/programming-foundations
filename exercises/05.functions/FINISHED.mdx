# Functions

<EpicVideo url="https://www.epicweb.dev/tutorials/programming-foundations/5-functions/break-functions" />

👨‍💼 Fantastic! You now know how to write type-safe functions.

You learned:

- 🧱 **Function declarations** define named, reusable functions
- 📥 **Parameter types** specify what a function accepts
- 📤 **Return types** specify what a function produces
- 🔮 **Type inference** figures out return types automatically
- ✍️ **Explicit return types** document your intent and catch mistakes
- ➡️ **Arrow functions** for concise callbacks and implicit returns

🦉 **When to use each function form:**

| Context                   | Use                  | Example                           |
| ------------------------- | -------------------- | --------------------------------- |
| Top-level named functions | Function declaration | `function processOrder() { ... }` |
| Callbacks                 | Arrow function       | `items.map((n) => n * 2)`         |
| Single-expression funcs   | Arrow (implicit)     | `const double = (n) => n * 2`     |

```ts
// Function declaration for top-level functions
function processOrder(items: CartItem[]): OrderResult {
	// Complex logic...
}

// Arrow functions for callbacks
const doubled = items.map((item) => item.price * 2)

// Arrow function with implicit return
const isExpensive = (price: number): boolean => price > 100
```
