# For-Of Iteration

ğŸ‘¨â€ğŸ’¼ The traditional `for` loop works, but there's a cleaner way! The `for...of`
loop gives you each element directly, without managing an index.

```ts
// Traditional - more to keep track of
for (let i = 0; i < items.length; i++) {
	const item = items[i]
	console.log(item)
}

// for...of - cleaner!
for (const item of items) {
	console.log(item)
}
```
