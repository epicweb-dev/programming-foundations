JavaScript supports all the standard math operators:

- `+` addition
- `-` subtraction
- `*` multiplication
- `/` division

```ts
console.log(10 + 5) // 15
console.log(10 - 3) // 7
console.log(4 * 3) // 12
console.log(20 / 4) // 5
```
