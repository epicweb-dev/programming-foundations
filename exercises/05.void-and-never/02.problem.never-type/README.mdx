```ts
function fail(message: string): never {

```

TypeScript uses this for smart type narrowing. After calling a `never` function,
TypeScript knows the code won't continue:

```ts
function process(value: string | null): string {
	if (value === null) {
		fail('Value cannot be null')
		// TypeScript knows we won't reach here
	}
	return value.toUpperCase() // value is definitely string
}
```
